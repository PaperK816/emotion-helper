# 情绪疏导小助手 - 需求文档

## 1. 项目概述

情绪疏导小助手是一款运行在Windows电脑上的桌面应用，帮助用户记录和管理每日情绪状态，提供压力疏导对话功能。

**设计理念**：简洁实用、以用户为中心、隐私优先

**目标用户**：普通电脑用户（无需编程知识）

---

## 2. 核心功能需求

### 2.1 每日心情记录模块

#### 2.1.1 总体心情记录
- 用户每天可记录一次总体心情
- 记录形式：数字评分（1-10分）+ 表情选择 + 可选备注文字

#### 2.1.2 精细化时段记录
- 用户可按时间段（如每小时）记录心情变化
- 同样采用：数字评分 + 表情选择 + 可选备注
- **完全自愿原则**：不强求记录频率，想记就记，不记也不影响使用

#### 2.1.3 心情选项定义

| 评分 | 表情 | 心情标签 |
|------|------|----------|
| 1-2  | 😫   | 非常糟糕 |
| 3-4  | 😔   | 比较糟糕 |
| 5-6  | 😐   | 一般     |
| 7-8  | 🙂   | 比较愉快 |
| 9-10 | 😊   | 非常愉快 |

---

### 2.2 AI 每日情绪总结模块

#### 2.2.1 自动生成
- 每天固定时间（如晚上10点）自动生成当日情绪总结
- 总结内容包括：
  - 今日心情趋势分析
  - 情绪波动说明
  - 可能的影响因素推测
  - 温馨提示或小建议

#### 2.2.2 手动触发
- 用户可随时点击「生成今日总结」按钮
- 根据当天已有的记录即时生成

#### 2.2.3 总结规则（本地引擎）
- 基于预设的情绪分析规则
- 结合评分、表情、备注关键词进行分析
- 无需联网，保护隐私

---

### 2.3 压力疏导对话模块

#### 2.3.1 对话功能
- 用户可在情绪不佳时打开对话界面
- 自由输入文字与AI交流
- AI根据预设规则给予回应

#### 2.3.2 回应策略
- 共情回应：理解用户感受
- 引导提问：帮助用户梳理情绪
- 放松建议：提供简单的放松技巧
- 积极鼓励：传递正能量

#### 2.3.3 扩展性
- 当前使用本地规则引擎
- 预留接口，未来可接入真实AI服务

---

### 2.4 历史记录查看器

#### 2.4.1 列表视图
- 按日期倒序展示所有记录
- 显示每条记录的摘要信息（评分、表情、备注）

#### 2.4.2 详情查看
- 点击某条记录可查看完整信息
- 包括该日所有时段记录

---

### 2.5 心情日历视图

#### 2.5.1 日历展示
- 以月历形式展示每天的心情
- 每天格子显示：评分 + 表情

#### 2.5.2 交互功能
- 点击某天可查看当日详细记录
- 支持月份切换

#### 2.5.3 颜色标识
- 不同心情等级用不同深浅的颜色标注
- 方便快速识别情绪趋势

---

## 3. UI/UX 设计要求

### 3.1 设计风格
- 简洁、清爽、不花哨
- 以功能为导向
- 大字体、大按钮，易于操作

### 3.2 配色方案
- **主色调**：淡黄色 (#FFF9E6 或类似)
- **辅助色**：柔和的灰色系
- **强调色**：温暖的橙色或绿色

## 4. 技术约束

### 4.1 数据存储
- 本地存储，不上传任何数据
- 使用 SQLite 数据库
- 数据定期自动备份到本地文件

### 4.2 隐私保护
- 所有数据存储在本地
- 不收集任何用户个人信息
- 不联网（除非用户主动选择接入AI服务）

### 4.3 系统要求
- 操作系统：Windows 10/11
- Python 3.10+
- 内存占用尽量低
